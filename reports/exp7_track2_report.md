# Отчет по эксперименту 7 (Индивидуальный трек 2)
**Предобуславливание усеченного Ньютона и оценка Хатчинсона**

## 1. Постановка задачи
Ускорить внутренний CG в HFN за счет диагонального предобуславливателя, оцененного методом Хатчинсона через `hess_vec`.

## 2. Функции, данные, параметры, железо
- Оракул: `ExponentialLossL2Oracle`.
- Датасет: LIBSVM `a9a` (разреженный).
- Искусственное ухудшение обусловленности:
  - первая половина признаков `*1000`,
  - вторая половина признаков `*0.001`.
- Сравнение:
  - базовый HFN,
  - HFN + preconditioned CG с Хатчинсоном.
- Параметры: `N` (число проб в Хатчинсоне) `<...>`.
- Железо: CPU `<...>`, RAM `<...>`.

## 4. Выводы
- Предобуславливание уменьшает число внутренних итераций CG на плохо обусловленных данных.
- Существует компромисс по `N`: маленькое `N` — шумная оценка диагонали, большое `N` — слишком дорого по `hess_vec`.
- Оптимальное `N` выбирается по минимальному итоговому времени при приемлемой скорости сходимости.
